# 🌍 100 Language Translation Bot

AI Translation Bot built using Python and Streamlit that can translate text into 100+ languages instantly.

## 🚀 Features

- Translate 100+ languages
- Select source and target language
- Fast translation
- Copy translated text
- Download translated text
- Voice output support
- Simple and beautiful UI

## 🛠 Technologies Used

- Python
- Streamlit
- deep-translator
- gTTS

## 📦 Installation

Install required libraries:

```bash
pip install streamlit
pip install deep-translator
pip install gtts
pip install pyperclip
```

## ▶ Run Project

```bash
python -m streamlit run app.py
```
